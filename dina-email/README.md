# Dina Email Module

## Overview
The `dina-email` module is a dedicated service for handling all email communications within the Dina Loan Application. It is responsible for sending notifications to users and staff regarding loan statuses.

## Technologies
*   **Mail API**: JavaMail 1.4.7
*   **Container**: EJB

## Features
*   **Notification Service**: Sends automated emails for loan submissions, approvals, and status updates.
*   **Templating**: Constructs email content based on defined templates (if applicable).
